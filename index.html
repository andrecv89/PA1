HTML,CSS y JavaScript:

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas.
HTML se encarga de la estructura y el contenido, CSS del diseño y la presentacion y JavaScript de la interactividad y la funcionalidad.

Definiciones:

HTML(HyperText Markup Language)

es el lenguaje de marcado que define la estructura y el contenido de una página web. Utiliza etiquetas para delimitar diferentes elementos, como parrafos, encabezados, imagenes, enlaces,etc.

CSS(Cascading Style Sheets)

es el lenguaje de estilo que se utiliza para controlar la presentacion de una pagina web. Permite definir aspectos como colores, fuentes, tamaños, margenes, etc. de los elementos HTML.

JavaScript(Program Language)

es un lenguaje de programacion que se utiliza para agregar funcionalidad interactiva a una pagina web. Permite manipular el contenido HTML, responder a eventos del usuario, realizar calculos, etc.
